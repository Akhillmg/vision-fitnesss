# Vision Fitness Mobile App

This is the mobile companion app for Vision Fitness, built with React Native and Expo.

## Getting Started

1. `npm install`
2. `npm start` - runs the Expo development server.
3. Scan the QR code with Expo Go on your phone.

## Integration

This app will connect to the backend API running at `http://localhost:3000`.
Ensure your computer and phone are on the same network.
